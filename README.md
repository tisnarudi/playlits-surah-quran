# playlits-surah-quran
Playlists audio Surah Al Fatihah, Al Falaq, An Naas, Al Ikhlas, Al Baqarah Ayat 255 (Ayat Kursi), dan Al Baqarah Ayat 285 - 286 disertai transliterasi latin dan terjemahan.
